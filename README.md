# 07Flip - GE Flip Finder

A RuneLite plugin that displays live Grand Exchange flipping data from [07flip.com](https://07flip.com) directly in your RuneLite sidebar.

## Features

- **Top Flips**: View the 10 most profitable GE flip opportunities, updated in real-time
- **Merch Alerts**: Get notified of conviction-tier merch opportunities with price targets
- **One-Click Access**: Click any item to view detailed analysis on 07flip.com
- **Configurable Refresh**: Set your preferred data refresh interval (60-600 seconds)

## Installation

This plugin is available through the [RuneLite Plugin Hub](https://runelite.net/plugin-hub).

1. Open RuneLite
2. Click the wrench icon to open Configuration
3. Click "Plugin Hub" at the bottom
4. Search for "07Flip"
5. Click Install

## Usage

Once installed, click the 07Flip icon in the RuneLite sidebar to view:

- **Flips Tab**: Current top 10 flip opportunities with buy/sell prices, profit, and ROI
- **Merch Alerts Tab**: Active merch detector alerts with price targets and hold times

Click any item row to open its detailed page on 07flip.com.

## Configuration

Access plugin settings via the RuneLite configuration panel:

- **Refresh Interval**: How often to fetch new data (default: 60 seconds)
- **Show Members Items**: Include members-only items in the flips list
